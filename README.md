iBank Skylight - UI Prototype 🏦
iBank Skylight is a high-fidelity Flutter UI kit designed to demonstrate a modern, intuitive banking experience. This project focuses on complex layout construction, custom widget design, and seamless screen transitions using Flutter's named routing system.

🎯 Project Goal
The primary objective of this project was to build a pixel-perfect banking interface that handles various financial workflows—from authentication to transaction management—while maintaining a clean and professional aesthetic.

✨ UI Features
Modern Design Language: Utilizes a custom color palette (Primary Purple) and the 'Satoshi' font family for a premium look.

Dynamic Dashboard: A functional grid-based service menu that links to every major feature of the app.

Custom Reusable Widgets:

ServiceItem: A standardized grid tile for feature navigation.

Account Cards: Visually distinct Visa/Account cards with balance displays.

Primary Buttons: Styled components used consistently across the app.

Interactive Navigation: A fully mapped-out "User Journey" using Flutter Named Routes, allowing for a clickable prototype experience.

📂 Screen Catalog
The prototype includes the following UI modules:

Authentication: Sign-In, Sign-Up, and Password recovery screens.

Financial Hub: Main Dashboard, Account Details, and Card Management.

Payments & Bills: Specialized layouts for Electric and Internet bill payments.

Transactions: Mobile top-up, Transfer, Withdraw, and Exchange interfaces.

Support & Settings: A chat-based customer support UI and settings for language and branch locations.

🛠 Tech Stack
Framework: Flutter (Material 3 enabled)

Language: Dart

Assets: Custom .png, .webp iconography and user avatars 

Architecture: Organized folder structure separating core constants, models, widgets, and screens.

🚀 How to Navigate
Start: The app launches into the Sign-In screen.

Explore: Use the "Login" button to reach the Home Dashboard.

Interact: Click on any grid item (like "Transfer" or "Pay the Bill") to navigate to the corresponding functional UI.
